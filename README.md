# Dali Library — Deprecated

**The Dali library has been deprecated** and is no longer maintained as a standalone project.

All functionality has been migrated to [**AeroDB**](https://github.com/microbian-systems/AeroDB) — the unified document-database and persistence framework for .NET.

Please update your references and dependencies to use AeroDB going forward. Migration guidance is available in the repository.

---

### Migration Steps

1. Replace `Dali` package references with the corresponding `AeroDB` packages
2. Update your `using` statements and namespace imports
3. Refer to the [AeroDB documentation](https://github.com/microbian-systems/AeroDB) for any API changes
